# Personal Registry

**Personal Registry** es una aplicación de Android moderna y fácil de usar diseñada para el seguimiento del peso corporal. Permite a los usuarios registrar su progreso diario, visualizar tendencias mediante gráficos y personalizar su experiencia según sus preferencias.

## 🚀 Características

- **Registro de peso:** Interfaz intuitiva para añadir y gestionar registros de peso diarios.
- **Visualización de datos:** Gráficos dinámicos para observar el progreso a lo largo del tiempo.
- **Gestión de historial:** Consulta, edita o elimina registros anteriores con facilidad.
- **Configuración personalizada:**
  - Soporte para unidades Métricas (kg) e Imperiales (lb).
  - Recordatorios de notificaciones personalizables (diario, cada 3 días, semanal).
  - Soporte para temas Claro, Oscuro y Adaptativo (Sistema).
- **Interfaz Moderna:** Desarrollada íntegramente con Jetpack Compose y siguiendo las líneas de diseño de Material Design 3.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** Kotlin
- **UI:** Jetpack Compose, Material 3
- **Persistencia de Datos:** Room Database (para registros) y DataStore Preferences (para ajustes).
- **Gráficos:** Compose Charts.
- **Arquitectura:** MVVM (Model-View-ViewModel).

## 📦 Requisitos e Instalación

1. **Requisitos:** Dispositivo con Android 13 (API 33) o superior.
2. **Instalación:**
   - Clona este repositorio.
   - Abre el proyecto en **Android Studio**.
   - Sincroniza el proyecto con los archivos Gradle.
   - Compila y ejecuta en un emulador o dispositivo físico.
